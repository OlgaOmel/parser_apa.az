# parser_apa.az/economy

парсит и сохраняет в базу данных статьи из раздела "Экономика" новостного сайта apa.az

*   **парсинг ссылок**: обход всех страниц раздела для получения ссылок на все статьи
*   **парсинг контента**: извлечение заголовка, даты, автора, тегов и полного текста статей
*   **сохранение в MongoDB**: загрузка в коллекцию `articles`


## настройка
1. зависимости:
    ```bash
    pip install -r requirements.txt
    ```
2.  для MongoDB (только для `mongo.py`):
    *   создать `.env` в корне
    *   MONGO_URI=<ссылка>

## Использование
**обязательно по порядку**:

1.  **Парсинг ссылок** (выход: `apa_economy_articles.json`):
    ```bash
    python all_links.py
    ```
2.  **Парсинг статей** (выход: `parsed_articles_complete.json`):
    ```bash
    python parser.py
    ```
3.  **Загрузка в базу данных** (выход: коллекция 'articles'):
    ```bash
    python mongo.py
    ```
